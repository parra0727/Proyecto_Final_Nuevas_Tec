Sección 1

Contexto y Problema de Negocio

¿Qué perfil tienen los clientes con mayor potencial de conversión hacia la adquisición de un Certificado de Depósito a Término (CDT)?

Una entidad bancaria contrató una empresa de marketing para contactar telefónicamente a posibles clientes y determinar su interés en adquirir un CDT. La información recolectada se almacenó en un archivo CSV con 45.215 registros y 17 variables, combinando características demográficas, financieras y de historial de contacto.

Antes de cualquier análisis exploratorio o construcción de modelos, es necesario garantizar la integridad del dataset. Este informe documenta el proceso sistemático de limpieza aplicado en cinco etapas, con los resultados reales obtenidos al ejecutar el script de procesamiento.

Variable objetivo

La columna y indica si el cliente aceptó o no el CDT. Valores posibles: yes / no.

Alcance

Este documento cubre la etapa de limpieza. El análisis exploratorio y la modelización son fases posteriores y no forman parte del presente entregable.

Valores extremos (outliers)

Se generaron boxplots para las siete variables numéricas. Se identificaron tres casos con valores imposibles: una edad de 776 años, duraciones de llamada negativas (mínimo −1.389 segundos) y un valor de 275 contactos previos. Los outliers en balance y pdays se conservaron por ser válidos en contexto bancario.

# age > 100 — biológicamente imposible (máx detectado: 776)
data = data[data['age'] <= 100]

# duration <= 0 — llamada de duración negativa (mín: −1.389 s)
data = data[data['duration'] > 0]

# previous > 100 — valor extremo detectado: 275
data = data[data['previous'] <= 100]

Errores tipográficos en variables categóricas

El análisis de subniveles reveló variantes del mismo nivel escritas de forma inconsistente. Se normalizó capitalización y se unificaron abreviaciones columna por columna.

# Normalizar capitalización
for col in cols_cat:
    data[col] = data[col].str.lower()

# Unificar variantes
data['job'].str.replace('admin.', 'administrative')
data['marital'].str.replace('div.', 'divorced')
data['education'].str.replace('sec.', 'secondary')
data.loc[data['education']=='unk'] = 'unknown'
data.loc[data['contact']=='phone'] = 'telephone'
data.loc[data['contact']=='mobile'] = 'cellular'
data.loc[data['poutcome']=='unk'] = 'unknown'
